# 💰 Kassir Bot — Foydalanish bo'yicha yo'riqnoma

## 1. Kirish (Avtorizatsiya)

Botni ochganingizda **/start** bosing.
Bot telefon raqamingizni so'raydi — **"Отправить номер"** tugmasini bosing.
Raqamingiz tizimda ro'yxatdan o'tgan bo'lsa, siz kira olasiz.

---

## 2. Asosiy menyu

Smena ochiq bo'lganda quyidagi tugmalar ko'rinadi:

| Tugma | Vazifasi |
|-------|---------|
| 💵 **Оплата** | Mehmondan to'lov qabul qilish |
| 📤 **Расход** | Xarajat yozish |
| 🔄 **Обмен** | Valyuta almashtirish |
| 💰 **Баланс** | Joriy balansni ko'rish |
| 🔒 **Закрыть смену** | Smenani yopish |

Smena yopiq bo'lsa faqat bitta tugma ko'rinadi:
- **Открыть смену** — Smenani ochish

---

## 3. 💵 To'lov qabul qilish (Оплата)

1. **💵 Оплата** tugmasini bosing
2. **Xona raqamini** yozing (masalan: )
3. Beds24'dan mehmon topilsa — ro'yxatdan tanlang
4. **Summani** yozing (masalan:  yoki )
5. **Valyutani** tanlang: UZS / USD / EUR
6. **To'lov usulini** tanlang: Наличные / Карта / Перевод
7. **Подтвердить** tugmasini bosing

✅ To'lov yozildi!

---

## 4. 📤 Xarajat yozish (Расход)

1. **📤 Расход** tugmasini bosing
2. **Kategoriyani** tanlang (Еда, Уборка, Ремонт, Такси, Хозтовары, Другое)
3. **Summani** yozing

### Summa yozish formatlari:
| Yozuv | Natija |
|-------|--------|
|  | 50,000 UZS |
|  | 20 USD |
|  | 20 USD |
|  | 20 USD |
|  | 20 USD |
|  | 15 EUR |
|  | 15 EUR |
|  | 20 USD |

4. **Tavsifni** yozing (masalan: Taomlar uchun)
5. **Подтвердить** tugmasini bosing

⚠️ 500,000 UZS dan yuqori xarajatlar uchun eganing tasdiqi kerak bo'ladi.

---

## 5. 🔄 Valyuta almashtirish (Обмен)

Mehmon valyuta almashtirganda:

1. **🔄 Обмен** tugmasini bosing
2. **Qabul qilayotgan valyutani** tanlang (masalan: UZS — siz olayotgan pul)
3. **Qabul summani** yozing (masalan: )
4. **Berayotgan valyutani** tanlang (masalan: USD — siz berayotgan pul)
5. **Berish summani** yozing (masalan: )
6. Bot kursni ko'rsatadi — **✅ Подтвердить** tugmasini bosing

### Misol:
> Mehmon 0 berdi, siz 128,000 UZS berdingiz:
> - Qabul: USD → 10
> - Berish: UZS → 128,000

✅ Ikkita tranzaksiya yoziladi: +0 va -128,000 UZS

---

## 6. 💰 Balansni tekshirish

**💰 Баланс** tugmasini bosing.

Bot ko'rsatadi:
- Har bir valyutadagi joriy balans (UZS, USD, EUR)
- Operatsiyalar soni
- Xarajatlar soni
- Smena ochilgan vaqti

---

## 7. 🔒 Smenani yopish

1. **🔒 Закрыть смену** tugmasini bosing
2. Bot kutilgan balansni ko'rsatadi
3. **UZS summani** sanab yozing
4. **USD summani** sanab yozing (yo'q bo'lsa )
5. **EUR summani** sanab yozing (yo'q bo'lsa )
6. Bot farqni ko'rsatadi — **Закрыть смену** bosing

✅ Smena yopildi! Egaga avtomatik hisobot yuboriladi.
